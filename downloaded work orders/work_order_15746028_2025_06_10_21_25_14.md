SCOPE OF WORK

Task # 1 – Locate the existing Zone Controller

Work with the Store Manager to locate the Zone Controller
The existing Zone Controller is manufactured by "Advanced Network Devices" and will be labeled as such. It is usually located on top of or nearby to the audio stack
This SHOULD be in the Manager’s Office alongside the other audio equipment (music player, amp, etc)
If you cannot locate, check the backroom/receiving area, next to the network rack
Refer to the "Old Zone Controller" attachment for reference if you need help identifying




Task # 2 - Capture details of the existing setup

Before starting any work, you must take several photos of the existing setup, and send them to the Worldlink Engineer.This benefits you, as the Engineer will review and provide guidance, and will also make rolling back easier (if required)
Ensure the pictures are clear and easy to interpret, and clearly show the following connections
Speakers, from wall to amplifier
Zone Controller to amplifier
Mood Media player, to amplifier
Amplifier to amplifier (if applicable)
Text the pictures to the Worldlink Engineer:**********




Task # 3 - Install new Algo Paging Device

The new Algo Paging Device will need to be connected to the store's existing audio setup. It will replace the existing Advanced Network Devices box you located in Task # 1.
The existing device is connected to one or more amplifiers via an RCA cable. You will re-use this cable to connect the new Algo device.
There is a document attached to your Work Order called "Algo Wiring Guide".
There are 3 pages to this document, each showing a different possible configuration.
Find the page that matches your particular configuration, and follow the instructions to install the new Algo device.
If you're unable to locate or are unsure, please reach out to Worldlink for assistance
You should not make any changes to the amplifier connections unless instructed to by Worldlink
Once you have the Algo device connected, connect it to the network using the existing Ethernet cable and validate that it connects successfully
The blue status lights on the front will turn on and remain on until the device has fully powered on




Task # 4 - Test new Algo Paging Device

Before testing, check to ensure you can hear overhead music playing through the speakers
If you can't, ask the MOD to check that the speaker volume is turned up for the room you're in
If the volume is turned up and you still can't hear music, escalate to the Worldlink Engineer for assistance
Once the status lights on the front of the device switch off, press the "Reset" button on the front of the unitbriefly and the unit will announce its IP address over the overhead speakers.
If you still hear overhead music or now nothing at all, check your wiring connections and escalate to the Worldlink Engineer for assistance -**********
You must be able to hear the IP address being read out over the speakers before continuing to Task # 5.
Press the "Reset" button a second time to stop the read out




Task # 5 - Contact Petco NOC to configure Algo Paging Device

Confirm with the Worldlink Engineer that the Algo device is connected correctly, and that you successfully completed the testing in Task # 4
You will then be provided with a Phone # to call




*** DO NOT PROCEED TO TASK # 6 WITHOUT FIRST GETTING IN CONTACT WITH THE PETCO NOC ***

Installing the phones before getting approval from the NOC willnot save you time, it will only add additional delays




Task # 6 - Replace Cisco Phone(s)

Work with the Store Manager to locate each of the phones to be replaced
For each phone…
Identify the Cisco phone to be replaced
Make note of the Line the phone is assigned to
Where this is displayed will vary depending on the model of phone
Examples: MANAGER, GROOMING, Vet 2
Make note of any existing data connections / “daisy-chained” devices
Each phone will have atleast 1 connection providing data and power to the unit
Some units may have a second connection to a nearby workstation or printer
Disconnect the Cisco phone and put it off to the site
Unbox the new Poly phone and assemble (see assembly guide attached to your Work Order)
To determine WHICH phone to install, compare the MAC address against the locations listed in the "High Level Overview" section
If there are no MAC addresses listed, pick one of the phones. After it boots up it will display either "Customer_Service" or "Grooming" on the screen. If that does not match the location you're installing, swap it with the other phone.
---
Install the desk stand
Install the handset
Connect the new Poly phone to the existing data run
There are two RJ-45 jacks on the back of the phone. The port on the leftconnects to the wall, and the port on the rightconnects to a “daisy-chained” device (if applicable)
Validate that the phone powers on and is receiving a network connection
!!! If prompted to set a password, use the Store Number (add a leading zero if the Store Number is only 3 digits - ex. Store #123 would be 0123)
Once online, the display of the phone should indicate which location this phone is assigned to ("Customer_Service" or "Grooming"). If that location doesn't match where you're currently installing the phone, swap it with the other one.
If the phone comes up and does not display the Store # and location ("Customer_Service" or "Grooming"), unplug the phone for 10 seconds and plug it back in
Once you have validated that the phone is fully functional, move on to the next phone
** Take a picture of the phone, screen, and MAC address sticker prior to moving on. This is one of your deliverables
** If there was an additional device daisy-changed to the phone (workstation, printer), validate functionality with the Store Manager
Remove the old Cisco phone from the site and dispose of properly
